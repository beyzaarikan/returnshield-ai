## Frontend'i Çalıştırma

Docker'a gerek yok. `frontend/` klasörüne gir, `index.html`'i tarayıcıda aç.

Canlı önizleme için (opsiyonel):

```bash
# Python varsa:
python -m http.server 3000

# Node varsa:
npx serve .
```

Sonra `http://localhost:3000` aç.

API bağlantısı için backend'in çalışıyor olması lazım (`http://localhost:8000`). Backend çalışmıyorsa `js/api.js` otomatik olarak mock data'ya geçer, çalışmaya devam edersin.

## Data / ML Dataset Setup

Raw dataset files are not included in this repository. Large raw datasets, processed data, and generated outputs are excluded from Git via `.gitignore`.

To run the notebooks, download the required datasets and place them under the paths below.

### 1. UCI Online Retail Dataset

Download from:

```text
https://archive.ics.uci.edu/dataset/352/online+retail
```

Place the dataset file under:

```text
data/raw/
```

This dataset is used for online retail data preparation, return/cancel proxy label creation, and baseline risk modeling.

### 2. ModCloth and RentTheRunway Clothing Fit Dataset

Download from:

```text
https://cseweb.ucsd.edu/~jmcauley/datasets.html#clothing_fit
```

Place the files as:

```text
data/raw/clothing_fit/modcloth_final_data.json
data/raw/clothing_fit/renttherunway_final_data.json
```

These datasets are used to generate item-level fit, length, quality, and review-based auxiliary signals.

### Generated Files

The following folders are generated by notebooks and are not tracked by Git:

```text
data/processed/
outputs/
```

Run the notebooks in this order:

```text
01_online_retail_data_preparation.ipynb
02_online_retail_baseline_modeling.ipynb
03_fit_signal_poc.ipynb
```
